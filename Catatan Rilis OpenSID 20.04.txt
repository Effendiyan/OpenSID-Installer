Lengkapnya, isi rilis opensid-installer versi 20.04 adalah sebagai berikut:
1. Set default stricton menjadi 'TRUE' agar menerima inputan default value 'NULL'.
2. Set default port 3606.
3. Set default max_execution_time(0) dan memory_limit(-1) saat menjalankan aplikasi.
4. Perbaikan view menggunakan assets bawaan opensid.
5. Perbaikan view logo menjadi center.
6. Sederhanakan pemanggilan view.
7. Penambahan view icon pd input.
8. Gunakan session menggati tag input hidden.
9. Perbaikan saat input pengguna menjadi id dinamis.
10. Penyesuaian input data profil desa.
